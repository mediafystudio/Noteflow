Associar arquivos .note ao Noteflow

Para associar arquivos .note ao Noteflow e usar o ícone personalizado, siga as instruções para seu sistema operacional:

Windows

  1. Baixe o arquivo Noteflow-icon.ico para seu computador
  2. Clique com o botão direito em qualquer arquivo .note
  3. Selecione "Abrir com" > "Escolher outro aplicativo"
  4. Selecione o navegador que você usa para acessar o Noteflow
  5. Marque a opção "Sempre usar este aplicativo para abrir arquivos .note"
  6. Clique em "OK"

Para associar o ícone:

  1. Abra o Registro do Windows (digite regedit no menu Iniciar)
  2. Navegue até HKEY_CURRENT_USER\Software\Classes\.note
  3. Crie uma nova chave chamada DefaultIcon
  4. Defina o valor padrão para o caminho completo do arquivo Noteflow-icon.ico

macOS

  1. Clique com o botão direito em um arquivo .note
  2. Selecione "Obter Informações"
  3. Na seção "Abrir com:", selecione seu navegador
  4. Clique em "Alterar tudo..."
  5. Confirme a alteração

Para associar o ícone no macOS, você precisará criar um aplicativo personalizado usando o Automator.

Linux

  1. Crie um arquivo .desktop em ~/.local/share/applications/
  2. Adicione as seguintes linhas (ajuste conforme necessário):
[Desktop Entry]
Type=Application
Name=Noteflow
Exec=seu_navegador %u
Icon=/caminho/para/Noteflow-icon.ico
MimeType=application/noteflow;
Terminal=false
Categories=Utility;

Em seguida, crie um arquivo MIME em ~/.local/share/mime/packages/noteflow.xml:

<?xml version="1.0" encoding="UTF-8"?>
<mime-info xmlns="http://www.freedesktop.org/standards/shared-mime-info">
  <mime-type type="application/noteflow">
    <comment>Noteflow Note</comment>
    <glob pattern="*.note"/>
  </mime-type>
</mime-info>

Execute update-mime-database ~/.local/share/mime para atualizar.

Nota: Estas instruções podem variar dependendo da versão específica do seu sistema operacional. Se encontrar dificuldades, consulte a documentação do seu sistema.
